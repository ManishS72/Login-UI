# Flutter Login UI

A simple and responsive Login UI built with Flutter.

## Features

- 📧 Email or Phone Login
- 🔒 Password Field (Hidden)
- 👁️ Show/Hide Password Icon
- 🔑 Forgot Password Button
- 🔵 LinkedIn Login Button
- ⚫ GitHub Login Button
- 📱 Responsive UI
